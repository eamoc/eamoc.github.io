--- layout: post title: Tango, didn't last long date: '2014-07-11T06:45:00.000-07:00' author: "@eamoc" tags: modified_time: '2019-01-28T03:16:36.590-08:00' blogger_id: tag:blogger.com,1999:blog-6955428370236550107.post-192429641632528716 blogger_orig_url: https://eamoc.blogspot.com/2014/07/tango-didn-last-long.html --- So I migrated over to Tango Studio from Gentoo. This seemed like a good idea because Debian is uncrashable basically, but of course it doesn't necessarily come with the latest versions certain files and libraries.
This is a problem when I tried to install Bitwig Studio, the first decent DAW I have come across that is installable on Linux.
Trying to run it throws an error:

/lib/x86_64-linux-gnu/libc.so.6: version `GLIBC_2.14' not found (required by /usr/bin/bitwig-studio)

This is because Debian only has version 2.13. After several attempts to install a later version of libc, I gave up. i fell in to dependency hell, and eventually my machine just wouldn't work. So I have capitulated and installed UbuntuStudio.
Despite my misgivings about it, It is so much easier to use, and, most importantly had the sound stack nicely configured. This is a MARE to sort out yourself.
Bitwig installed without a hitch.
