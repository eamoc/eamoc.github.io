If [[ -z $DISPLAY ]] && [[ $(tty) = /Dev/tty ]]; then exec startx; fi

Only console users are allowed to run the x server
To ensure startx runs on login, copy the following in to bash_profile:
if [[ ! $DISPLAY && $XDG_VTNR -eq 1 ]]; then exec startx fi
